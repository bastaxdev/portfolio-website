# bastaxdev — Portfolio

Personal portfolio website built with Next.js, TypeScript and Tailwind CSS.

## 🚀 Live

[bastaxdev.com](https://www.bastaxdev.com)

## Tech Stack

- Next.js 15, TypeScript, Tailwind CSS
- Framer Motion
- Deployed on Vercel

## Run locally
```bash
npm install
npm run dev
```
